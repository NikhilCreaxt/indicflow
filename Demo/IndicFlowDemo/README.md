# IndicFlow Demo

Minimal Unity project showing Hindi/Devanagari shaping with `TMPro.HarfBuzzTextMeshProUGUI`.

## Open

1. Open `Demo/IndicFlowDemo` in Unity.
2. Open scene: `Assets/Scenes/SampleScene.unity`.
3. Press Play and use the toggle button to switch TMP vs HarfBuzz TMP.

## Package Source

This demo uses the local package in this repository:

- `../../../Packages/com.unity.ugui`
